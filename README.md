# Enterprise Dashboard

A **modern enterprise dashboard** built with **React, TypeScript, Redux Toolkit, and Vite**.
It provides **real-time analytics, optimized performance, and scalable architecture**.

---

## 🚀 Features

* Real-time analytics with **WebSocket**
* Advanced state management with **Redux Toolkit**
* Interactive charts using **Recharts**
* Optimized performance with **React.memo, useMemo**
* Feature-based architecture
* Unit & E2E testing (**Jest, Cypress**)

---

## 🛠 Tech Stack

* React 18
* TypeScript
* Redux Toolkit
* React Router
* Recharts
* Vite
